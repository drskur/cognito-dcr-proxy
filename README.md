# @drskur/dcr-proxy

MCP-compatible OAuth proxy for Amazon Cognito, packaged as an AWS CDK construct.

Lets MCP clients (Claude Code, Kiro CLI, MCP Inspector, ...) talk to a Cognito User Pool via Dynamic Client Registration while keeping the Cognito Hosted UI flow intact.

> Status: early development. Not yet published to npm.

## Install

```sh
pnpm add @drskur/dcr-proxy
```
